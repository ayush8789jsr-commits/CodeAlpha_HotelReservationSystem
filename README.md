# Hotel Reservation System

A Java-based Hotel Reservation System that allows users to view rooms, book rooms, cancel bookings, and manage room availability.

## Features
- View available rooms
- Book a room
- Cancel booking
- Room status tracking

## Technologies Used
- Java
- ArrayList
- Object-Oriented Programming (OOP)
